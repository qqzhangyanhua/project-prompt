# Product Overview

PromptHub is an AI prompt sharing platform designed for AI enthusiasts, content creators, and developers. It serves as a discovery and sharing hub for high-quality AI prompts.

## Core Features
- **Smart Browsing**: Card-based layout with latest/popular sorting
- **Precise Filtering**: Multi-category filtering and keyword search  
- **Interactive Experience**: Like, favorite, and one-click copy functionality
- **Content Publishing**: Streamlined publishing workflow with tag management
- **User Center**: Personal profile showcasing published and favorited content

## Target Users
- AI enthusiasts and researchers
- Content creators using AI tools
- Developers working with AI APIs
- Anyone looking for quality AI prompts

## Key Value Propositions
- Curated high-quality AI prompts
- Simple, intuitive user interface
- Community-driven content sharing
- Efficient prompt discovery and management
